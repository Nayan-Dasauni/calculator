# Calculator

A simple calculator built with **HTML**, **CSS**, and **JavaScript**. It supports input through both the keyboard and mouse, so users can either click the on-screen buttons or type directly from the keyboard.[1][2][3]

## Features

- Perform basic arithmetic operations: addition, subtraction, multiplication, and division.[3]
- Accept input from both keyboard and mouse.[1][4][3]
- Clear the full display using `AC` or the `Escape` key.[3]
- Delete the last entered character using `DEL` or the `Backspace` key.[3]
- Calculate the result using `=` or the `Enter` key.[1][3]
- Built using only HTML, CSS, and JavaScript with no external libraries.[5][3]

## Technologies Used

- HTML for the structure of the calculator.[5]
- CSS for styling and layout.[5][6]
- JavaScript for button handling, keyboard input, and calculation logic.[5][6]

## How to Use

1. Clone the repository:

```bash
git clone https://github.com/Nayan-Dasauni/calculator.git
```

2. Open the project folder.
3. Run the `index.html` file in your browser.
4. Use your mouse to click buttons or your keyboard to type numbers and operators.[4][3]

## Keyboard Controls

- `0-9` → Enter digits.[3]
- `+`, `-`, `*`, `/` → Perform arithmetic operations.[3]
- `.` → Add a decimal point.[3]
- `Enter` → Calculate the result.[1][3]
- `Backspace` → Delete the last character.[3]
- `Escape` → Clear the display.[3]
