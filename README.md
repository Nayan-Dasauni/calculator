# Calculator

A simple calculator built with **HTML**, **CSS**, and **JavaScript**. It supports input through both the keyboard and mouse, so users can either click the on-screen buttons or type directly from the keyboard. 

## Features

- Perform basic arithmetic operations: addition, subtraction, multiplication, and division.
- Accept input from both keyboard and mouse. 
- Clear the full display using `AC` or the `Escape` key. 
- Delete the last entered character using `DEL` or the `Backspace` key. 
- Calculate the result using `=` or the `Enter` key.
- Built using only HTML, CSS, and JavaScript with no external libraries.

## Technologies Used

- HTML for the structure of the calculator.
- CSS for styling and layout.
- JavaScript for button handling, keyboard input, and calculation logic.

## How to Use

1. Clone the repository:

```bash
git clone https://github.com/Nayan-Dasauni/calculator.git
```

2. Open the project folder.
3. Run the `index.html` file in your browser.
4. Use your mouse to click buttons or your keyboard to type numbers and operators.

## Keyboard Controls

- `0-9` → Enter digits.
- `+`, `-`, `*`, `/` → Perform arithmetic operations.
- `.` → Add a decimal point.
- `Enter` → Calculate the result.
- `Backspace` → Delete the last character.
- `Escape` → Clear the display.
